## 本周总结

本周介绍了六种常用的`字符串分析算法`，从简单到难分别是：

- 字典树
  - 大量高重复字符串的存储与分析
- KMP
  - 在长字符串里找模式
- Wildcard
  - 带通配符的字符串模式（弱一点的正则）
- 正则
  - 字符串通用模式匹配（在理论等价于有限状态机）
- 状态机
  - 通用的字符串分析
- LL LR
  - 字符串多层级结构分析

然后学习了其中的三种：字典树、KMP 和 Wildcard
